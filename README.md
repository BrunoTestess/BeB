# Bruno & Ana Beatriz 🤍

Site minimalista criado como presente romântico.

## Estrutura

```
romance-bruno-ana/
├── index.html        # Página principal
├── css/
│   └── style.css     # Estilos
├── js/
│   ├── counter.js    # Contador de dias/meses/horas
│   └── gallery.js    # Galeria de fotos
├── img/              # Coloque fotos fixas aqui (opcional)
└── README.md
```

## Como rodar

Abra o `index.html` direto no navegador — não precisa de servidor.

Para uma experiência melhor no VS Code, instale a extensão **Live Server**
e clique em "Go Live" no canto inferior direito.

## Como personalizar

- **Nomes / data**: edite `index.html` nas seções `hero` e `counter-section`
- **Momentos da linha do tempo**: edite os `.timeline-item` no `index.html`
- **Carta**: edite o `<blockquote class="letter">` no `index.html`
- **Fotos fixas**: coloque imagens na pasta `img/` e referencie no HTML
- **Cores**: todas as cores estão em `css/style.css`

## Publicar online (GitHub Pages)

1. Suba o projeto no GitHub
2. Vá em Settings → Pages
3. Escolha a branch `main` e salve
4. O site fica disponível em `https://seuusuario.github.io/romance-bruno-ana`
